Documentação rápida

Porta padrão: 9999 (configurável por PORT).

Sem dependências externas no Node.js (usa somente módulos nativos).

Instalação

cp .env.example .env
npm start

Termux

pkg update -y
pkg install nodejs git -y
git clone <seu-repo>
cd Server-ia
cp .env.example .env
npm start

Deploy Render

Build Command: npm install
Start Command: npm start
Environment: PORT=9999 + FIREBASE_DATABASE_URL (+ FIREBASE_DATABASE_SECRET opcional)

Rotas principais

  • GET /api/services
  • POST /api/services/:id/status
  • POST /api/users/credit
  • POST /api/games/crash/round
  • POST /api/games/crash/bet
  • POST /api/games/crash/cashout
  • GET /api/stats
  • GET /api/logs
  • GET /events (SSE)
